Step 1 — Connect Your Firebase Database
Do this first — your audience data, tool results, and subscribers go here. You own it from day one.
Not connected
1
Go to console.firebase.google.com → create a new project (the free Spark plan is fine)
2
In your project: click Firestore Database → Create database → choose a region → Start in test mode
3
Click Authentication → Get started → Sign-in method → enable Email/Password
4
Go to Project Settings → scroll to "Your apps" → Add a web app → copy the firebaseConfig and paste it below
project-id Your audience data writes to this Firebase project
Firestore security rules — paste these in your Firebase console
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /subscribers/{doc} {
      allow create: if true;
    }
    match /toolUsage/{doc} {
      allow create: if true;
    }
    match /toolResults/{doc} {
      allow create: if true;
      allow read: if true;
    }
    match /supportRequests/{doc} {
      allow create: if true;
      allow read, update: if request.auth != null;
    }
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
In Firebase Console → Firestore → Rules → paste the above → Publish. This lets your landing page collect emails, log tool usage, and save tool results without requiring login.
