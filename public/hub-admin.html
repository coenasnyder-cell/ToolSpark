project-id Your audience data writes to this Firebase project
Firestore security rules — paste these in your Firebase console
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /subscribers/{doc} {
      allow create: if true;
    }
    match /toolUsage/{doc} {
      allow create: if true;
    }
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
In Firebase Console → Firestore → Rules → paste the above → Publish. This lets your landing page collect emails and log tool usage without requiring login.
