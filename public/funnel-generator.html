Fastest
Netlify Drop
Go to drop.netlify.com, drag your HTML file onto the page, and you'll have a live URL in seconds. Free, no account required.
Easiest
Carrd
Create a free account at carrd.co, start a new site, choose "Embed" → "Code", paste your HTML in, and publish. Paid plan (~$19/yr) needed for a custom domain.
Already have a site?
Squarespace / Webflow
In Squarespace, add a new page and use a Code Block to paste your HTML. In Webflow, use the Embed element. Both work — Webflow gives more control over layout.
